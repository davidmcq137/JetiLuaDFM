DFM-TimV

The Variable Rate Timer App (DFM-TimV) creates two count-down timers. Both are controlled by the value of a proportional stick (typically throttle). One timer (the Variable Rate Timer) goes at normal speed with the stick at 100% and slows down proportionally (or with optional expo) as the stick is reduced. The other timer (the Variable Duration Timer) always counts down at normal speed but computes the time to zero based on the value of the stick. Both timers reach the final 0:00 time simultaneously.

This app is intended as a substitute for actual capacity measurements of battery mAh or fuel (gas or jet).

Version 0.4

Documentation

Usage

  1. Assign the main control (e.g. Throttle) and make sure to set it to proportional
  2. Assign switches to start/stop and reset the timer (optional)
  3. Set the starting time for the countdown on the main app menu
  4. Choose a countdown for the final seconds of the Variable Duration Timer. If set to n seconds, the app will announce each of the last n seconds as it counts to 0:00
  5. You can optionally set up stick shakes at t=0
  6. There is a setting for a min throttle value with the stick at idle to compensate for idle fuel flow on non-electric models
  7. The app creates one double telemetry window, enable it with the Displayed Telemetry menu command on the TX. The Variable Rate timer is on top and the Variable Duration timer is on the bottom
  8. In the telemetry window, the smaller numbers on the right are the throttle stick percentages (0 to 100). The top value is the actual stick position, the bottom number is after expo is applied.
