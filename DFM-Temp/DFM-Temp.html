DFM-Temp

For display of temperatures including images of the engines and placement of temperature probes.

Version 1.0 26-Jan-2023

Documentation

There is an instructional video by Harry Curzon on how to set up and use this app, check out Harry's Youtube channel

Usage

  1. Assign telemetry sensors to the temperature probe inputs
  2. Edit the probe parameters as required
  3. Select the type of display desired
