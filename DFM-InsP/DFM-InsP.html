DFM-InsP.lua

This is a companion app for the instrument panel creator available on our website. Once a pilot has created a set of panels, the panels along with this companion Jeti lua app can be downloaded using Jeti Studio with the unique key produced for your content on the website.

Version 0.89 Oct 27, 2023

Instrument Panel Display App

Once an instrument panel is defined on the website, it can be downloaded to the transmitter with the Lua App manager in Jeti Studio.

After installing the app on the transmitter, sensors, controls and lua expressions can be assigned to the gauges, lights and textboxes to animate them with live sensor information.

There are instructional vides by Harry Curzon on how to set up and use this app, check out Harry's Youtube channel. For some more detailed videos after you see Harry's, see the videos in a playlist available on Dave's channel.
